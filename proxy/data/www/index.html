Before you use this network you need to install a security certificate. This is a one-time step — once done your browser will trust all the local apps without any warnings.

Step 1 — Download the certificate file  ↓
  • 2Open your Downloads folder and double-click the file you just downloaded.
  • 3A window will appear. Click Install Certificate…
  • 4Select Local Machine then click Next. If Windows asks for permission, click Yes.
  • 5Choose Place all certificates in the following store, then click Browse and select Trusted Root Certification Authorities.
  • 6Click Next then Finish. You're done — close and reopen your browser.
Step 1 — Download the certificate file  ↓
  • 2Open your Downloads folder and double-click the file. An app called Keychain Access will open.
  • 3Find Hippity Internal CA in the list and double-click it.
  • 4Click the arrow next to Trust to expand it, then change When using this certificate to Always Trust.
  • 5Close the window. Mac will ask for your password — enter it and click Update Settings. Done.
Step 1 — Download the certificate file  ↓
  • 2Your phone will say "Profile Downloaded" at the top of the screen. Open the Settings app.
  • 3Near the top of Settings you'll see a new option — Profile Downloaded. Tap it, then tap Install and enter your passcode.
  • 4Go back to Settings → General → About → Certificate Trust Settings (scroll to the very bottom of About).
  • 5Turn on the switch next to Hippity Internal CA. Tap Continue to confirm. Done.
Step 1 — Download the certificate file  ↓
  • 2Open the Settings app and search for certificates in the search bar at the top.
  • 3Tap Install from storage (it may say Install from device storage or similar).
  • 4Find and tap the file you downloaded. When asked what it's for, choose CA Certificate.
  • 5Give it a name like Hippity and tap OK. Done.

The exact steps vary a little depending on your phone brand — Samsung, Pixel, and others may look slightly different, but the options will be the same.

Step 1 — Download the certificate file  ↓
  • 2Open a terminal and run:

    sudo cp ~/Downloads/hippity-ca.crt /usr/local/share/ca-certificates/
  • 3Then run:

    sudo update-ca-certificates
